In the Note Tree, it's possible to select multiple notes at one time. This won't affect the currently displayed note, but it allows to do batch operations.

Types of multiple selection

Selecting notes one by one

To individually select each note, press the Alt key and click on the note to be selected.

The selected notes will be indicated via a different background color and an icon.

Selecting notes in a sequence

When notes are in a sequence (e.g. one underneath the other), the entire selection can be made just by selecting the first and the last item instead of individually clicking all the notes.

To do so, first select the note to start the selection with. Then hold Shift and click on the note to end the selection with. All the notes between the start and the end note will be selected as well.

Selected notes vs. active note

There is a distinction between selected notes and the active note:

  • Selected notes are simply the notes that were selected in the multiple section process (i.e. either via Alt or Shift).
  • The active note is simply the note that whose content is being displayed to the right of the tree.
